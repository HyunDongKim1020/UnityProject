  ## 💾 프로젝트 요약
##### -오픈 필드를 적극적으로 활용하여 플레이어의 자유도 강조.
##### -몬스터 별 난이도를 지정해 단순한 패턴 및 지루함 해소
##### -생존자/마피아 역할 별 기능 및 목표를 부여함으로 유저의 흥미 유도.
##### -포톤을 통한 채팅, 투표 기능 등의 멀티 시스템 기능을 군현함으로 친구나 가족과 함께 즐길 수 있음.
##### -온라인 멀티 시스템으로 싱글 게임과의 차별성 강조
