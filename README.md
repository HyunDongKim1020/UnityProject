## 💾2.프로젝트 기획의도
##### -판타지 게임을 좋아하다 보니 더 위쳐처럼 중세 판타지 배경으로 모험을 즐길수 있는 게임을 제작해 보고 싶었습니다.


## 💾2-1.프로젝트 주제
##### -오픈 월드에서 플레이어가 자유롭게 맵을 탐방하며 왕국에서 퀘스트를 받고 던전에서 사냥하며 최종적으로 던전의 보스 몬스터를 처치하고 국왕에게 보검을 하사 받는다.
